What are Transformations?

Transformations are functions acting on points. When transforming shapes, apply function to each point of the shape.

Translation

The most basic type of transformation is the translation.
Translate by simply adding matrix value to point.
x1 = x + tx
y1 = y + tx

Linear Transforms

Linear transformations are single transformations that apply only one transformation matrix or movement. They satisfy the following property:
T(u + v) = A(u + v) = Au + Av = T(u) + T(v)
T(cu) = A(cu) = cAu = cT(u)

Scale Shear Rotation Reflection
Uniform scale is applied by multiplying the same S on both x and y. A shear will keep all points on a given line L fixed while other points are shifted parallel to L by a distance proportional to their perpendicular position from L Create rotation matrix based on theta (θ) you wish to rotate point around the origin. A reflection transform will swap all pairs of points that are on exactly opposite sides of the line of reflection

Affine Transforms

Affine transformations combine both a linear transformation and a translation. It is a mapping method used to preserve points, straight lines, and planes. Sets of parallel lines will remain parallel after an affine transformation.
We combine both factors to create a 3 dimensional vector

1 0 tx
0 1 ty
0 0 1

Coordinate System Transform

Coordinate system transforms extend on the concept of affine transforms as all lines and points are preserved and only distorted.
